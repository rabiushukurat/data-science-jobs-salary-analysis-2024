# Data Science Jobs & Salary Analysis 2024

## Project Overview
Analysed 6,600 data science job records to uncover salary trends across 
job titles, experience levels, company sizes and countries worldwide.

## Tools Used
- Microsoft Excel (data cleaning, pivot tables, analysis)
- Tableau Public (data visualisation, dashboard)

## Key Insights
1. **Analytics Engineering Manager is the highest paying role** at $399,880 average salary
2. **Salary more than doubles from Entry to Executive level** — entry level starts at $84k, executive reaches $189k
3. **Qatar pays the highest salaries globally** at $300,000 average — higher than the US at $157k
4. **Medium sized companies pay more than large ones** — $149k vs $120k average
